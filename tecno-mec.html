Tecno-Mec Academy

"Tecnología para la vida"
Santo Domingo, Dominican Republic · 2020 – 2023

// The Mission

Tecno-Mec Academy was an educational robotics and technology program I founded in the Dominican Republic with a single goal: make engineering and technology accessible to children from an early age, through hands-on learning and real projects.

The academy's slogan — "Tecnología para la vida" (Technology for life) — reflected the belief that technical skills are not just for engineers, but a foundation for problem-solving in any field.

// What Students Learned

  • Robot assembly and mechanics using mBot Ranger kits — students built robots from scratch, learning about motors, sensors, and structural design
  • Programming fundamentals using Scratch, creating interactive 2D games and animations
  • 3D design and digital fabrication using Tinkercad and 3D printing
  • Basic electronics and circuit design using Rokenbok Education engineering kits
  • Virtual reality exploration — introducing immersive technology as a creative and educational tool

// Teaching Approach

Every session was project-based. Students didn't just learn concepts — they built something tangible by the end of each class. This hands-on approach made abstract ideas like circuits, algorithms, and mechanical design concrete and intuitive for children of all ages.

I designed all the curricula from scratch, adapting complexity to different age groups while keeping the core principle consistent: learn by doing.

// Why It Mattered

Running Tecno-Mec reinforced something important about how I approach engineering: the ability to explain a complex system simply is just as valuable as being able to build it. Teaching children to think like engineers — to break problems down, iterate, and not be afraid to fail — shaped how I communicate technical ideas with non-technical stakeholders today.
